# Portfolio Starter

Static React + TypeScript + Vite starter prepared for Cloudflare Pages.

## Commands

- `npm run dev`
- `npm run build`
- `npm run preview`

## Cloudflare Pages

- Build command: `npm run build`
- Output directory: `dist`

## Notes

- Uses `@headlessui/react` for accessible interactive UI.
- Keeps the site static by default.
